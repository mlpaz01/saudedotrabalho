# Template HSM: `boasvindas_plataforma`

**Uso:** primeiro contato da plataforma com o colaborador (chama atenção pra abrir conversa).

---

| Campo | Valor |
|---|---|
| **Nome** | `boasvindas_plataforma` |
| **Categoria** | `UTILITY` |
| **Idioma** | `pt_BR` |
| **Variáveis** | 1 (`{{1}}` = nome da empresa) |

## Corpo

```
Olá! Você está sendo cadastrado(a) na *Plataforma Saúde do Trabalho* pela *{{1}}*.

Por aqui você poderá:
✓ Responder pesquisas
✓ Acessar treinamentos
✓ Solicitar acolhimento
✓ Registrar denúncias com sigilo

Responda *MENU* a qualquer momento pra ver as opções.

_Responda SAIR pra parar de receber mensagens._
```

## Exemplo

- `{{1}}` → `MF Conexões`
